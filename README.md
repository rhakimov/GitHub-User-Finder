# GitHub User Finder

**Автор:** Хакимов Рамзан Ильфатович

## Описание

Приложение для поиска пользователей GitHub с возможностью добавления в избранное. Использует официальный GitHub API для получения информации о пользователях.

## Функциональность

* Поиск пользователей GitHub по имени
* Отображение основной информации: логин, имя, количество публичных репозиториев, количество подписчиков
* Добавление пользователей в избранное
* Сохранение избранных пользователей в JSON-файл
* Удаление пользователей из избранного

## Использование API GitHub

Приложение использует GitHub API v3:
* Базовый URL: `https://api.github.com`
* Эндпоинт для поиска пользователя: `GET /users/{username}`
* Ограничение запросов: 60 запросов в час для неавторизованных пользователей

## Установка и запуск

1. Установите Python 3.x
2. Установите зависимости:
   ```bash
   pip install requests
```
3. Запустите приложение:

```bash
git clone https://github.com/твой-логин/weather-diary.git
cd weather-diary
python main.py
```
